## 🧰 &nbsp; How to Install:

```
git clone git@github.com:rafnobrega/omdb-app.git omdb-app
```

```
npm install
```
Copy the `.env.example`, rename it to `.env`, and modify the following key with your OMDb key:
```
REACT_APP_OMDB_KEY
```
